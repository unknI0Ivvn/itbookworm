# ItBookworm

This project is a simple Go-based API to manage and search books.

## Getting Started
1. Clone the repository
2. Run `go run cmd/main.go`
3. Access the API at `http://localhost:8080/books`

## API Endpoints
- `GET /books`
- `GET /books?title={title}`

Настройте подключение к базе данных.
Реализуйте функции для работы с книгами в базе данных (например, добавление, обновление, удаление книг).
Создание CRUD API:
Create (POST): Добавление новой книги в базу данных.
Read (GET): Получение списка книг или поиска книг по названию/автору.
Update (PUT): Изменение данных о книге.
Delete (DELETE): Удаление книги из базы.
Для каждого действия добавьте отдельный обработчик HTTP-запросов.
2. Обработка ошибок и валидация данных

Валидация данных: Добавьте проверку входящих данных (например, проверку правильности формата ISBN, проверку обязательных полей, таких как название и автор).
